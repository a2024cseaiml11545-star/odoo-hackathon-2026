# TransitOps Fuel & Expense Management

## Overview
TransitOps Fuel & Expense Management is an Odoo module designed to manage vehicle fuel records, expenses, and reports efficiently.

## Features
- Fuel Log Management
- Vehicle Expense Tracking
- Fuel Cost Monitoring
- Expense Reports
- Search and Filter Options
- CSV/PDF Report Support
- User-Friendly Interface

- Views
- Reports
- Security
- Manifest File
